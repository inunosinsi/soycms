p_block:id="read_entry_ranking_list"内で使用できるcms:idタグ
cms:id タグ 説明
entry_id A 記事URLを出力します
title 全て 記事タイトルを出力します
count 全て 表示回数を出力します
※サムネイルプラグイン等のブロック内で使用できるプラグイン由来のcms:idを使用できます。

※b_block:id="entry"やb_block:id="entry_list"内で表示回数を出力する場合はcms:id="view_count"が使用できます
