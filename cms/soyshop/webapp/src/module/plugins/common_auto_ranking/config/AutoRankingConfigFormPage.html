テンプレートへの記述

<!-- shop:module="common.auto_ranking" -->
<section>
	<h1>ランキング</h1>
	<ul>
		<!-- block:id="ranking_item_list" cms:customfield="off" -->
		<li>
			<a cms:id="item_link"><img cms:id="item_small_image"></a>
			<p>
				<a cms:id="item_link*"><!-- cms:id="item_name" /--></a>
			</p>
		</li>
		<!-- /block:id="ranking_item_list" -->
	</ul>
</section>
<!-- /shop:module="common.auto_ranking" -->
※block:id="ranking_item_list"内では、block:id="item"やblock:id="item_list"で使用できるcms:idを使用できます。
※cms:customfield="off"にすることで、カスタムフィールドの拡張ポイントの実行をOFFにし、表示速度の高速化をはかります。
