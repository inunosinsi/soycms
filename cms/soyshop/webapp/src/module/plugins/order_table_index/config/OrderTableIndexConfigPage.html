このプラグインではデータをより高速に取得できるようにデータベース内の値を整理します。
注文件数が多い場合は数回(運営年数が長い場合は数十回)のデータの最適化の処理を必要とします。
処理の実行は管理画面にアクセスした時に自動で実行されます。

最適化をより速く終了したい場合は下記のコマンドをcrontabに登録して下さい。 
*/10 * * * *  2000
※時限実行の間隔はサーバのリソースに応じて適宜調整して下さい。

当プラグインは最適化終了後に自動でアンインストールされます

データベースの最適化の背景は下記の記事をご覧ください。
SOY Shopのデータベースのインデックスの見直しで運営の限界を上げる - saitodev.co
