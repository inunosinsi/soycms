カスタムサーチフィールドがインストールされていません

商品詳細ページで子商品の表示設定がされていません

商品一覧ページの設定

  • 商品一覧用のページを新たに作成し、商品一覧ページの設定でその他を選び、CustomSearchFieldChildListを選びます。
  • カスタムサーチフィールドで商品を選択する場合はhttp://ドメイン/{ショップID}/{URI}/{サーチフィールドID}/{フィールドの値}になります。

商品一覧ページで使用できるソート用リンクタグ


<a css:id="custom_search_sort__asc">の昇順</a>
<a css:id="custom_search_sort__desc">の降順</a>

商品一覧ページを開いた直後にソートを反映する方法

  • 一覧ページへのURLの末尾に?custom_search_sort=フィールドID&r=0を追加
  • rの値は0は昇順で1が降順になります

検索結果ページの設定

  • ページの追加から、検索結果ページを作成する。
  • 作成した検索結果ページの編集から、検索モジュールの設定を開き、Custom Search Field Child Listを選び、更新する。
  • 公開側ページにフォームを設置する
