変更しました
ページ毎に非同期カートボタンを使用するか設定します。

テンプレートへの記述例

<!-- block:id="item" -->

<form cms:id="async_cart_form"><!-- レガシーなブラウザ対策のために同期用のformタグも設置しておく -->

<select cms:id="async_cart_select"></select>
<a cms:id="async_cart_link">カートに入れる</a>
<input cms:id="standard_price_helper"><!-- 商品規格プラグインと併用するのであればこのタグは必須です -->

</form>

<!-- /block:id="item" -->

カートの中身の非同期更新の設定

  • 商品点数を非同期更新に対応する場合は、数字をid="soyshop_cart_item_count"を含んだタグで囲みます。
  • 小計を非同期更新に対応する場合は、数字をid="soyshop_cart_sub_total"を含んだタグで囲みます。
