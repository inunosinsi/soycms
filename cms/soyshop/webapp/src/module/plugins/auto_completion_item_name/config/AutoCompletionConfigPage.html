読み方の自動取得
 件
ご利用中のサーバにMecabをインストールすると、読み方の自動取得の機能が使用できます。

設定方法
検索モジュールプラグイン等の検索用のフォームを設置するプラグインで、 
<input id="auto_completion" type="text" name="q">
のようにid="auto_completion"を追加すると、自動でオートコンプリートで必要なライブラリを読み込み、使用できる状態になります。
